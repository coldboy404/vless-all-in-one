# vless-all-in-one

这是从上游项目整理后的 **代码仓库版**，仅保留脚本、配置与实现文件，去掉了原仓库里大量面向使用者的长篇说明文档，方便二次维护、审阅和继续开发。

## 仓库定位

本仓库主要用于：
- 保存核心部署脚本与相关配置代码
- 便于在 GitHub 上继续维护、审查和做二次改造
- 降低文档噪音，让代码结构更聚焦

## 当前保留内容

- Shell 脚本（如主安装脚本）
- JSON / YAML / TOML / conf 等配置文件
- 可能存在的 Python / JS / TS 辅助脚本
- `Dockerfile` / `Makefile` / `.gitignore` / `LICENSE`

## 未保留内容

以下内容没有从上游完整搬运：
- 大型使用说明文档
- 面向终端用户的冗长教程
- 非代码型展示资料
