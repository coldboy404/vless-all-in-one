# 功能摘要

以下内容不是照搬上游长文，而是提炼出的功能摘要，方便在 GitHub 上快速理解项目边界。

## 协议与传输

根据当前脚本头部和上游说明，项目覆盖的协议族包括：
- VLESS 系列
- VMess
- Trojan
- SOCKS5
- Shadowsocks 2022
- Hysteria2
- TUIC v5
- Snell v4 / v5
- AnyTLS
- NaïveProxy

## 典型能力

### 1. 多协议共存
同一台机器上安装多个协议实例，适合同时兼顾：
- TCP 场景
- UDP 场景
- CDN 场景
- 端口复用场景

### 2. 用户管理
上游说明中明确提到支持：
- 添加 / 删除用户
- 流量配额
- 到期时间
- 启用 / 禁用
- 分享链接与订阅
- 流量统计

### 3. 路由与分流
支持把不同目标流量送往不同出口，常见用途包括：
- 直连
- WARP
- 链式代理
- 多出口策略

### 4. 订阅服务
可生成或聚合多种客户端订阅配置，用于统一分发节点信息。

### 5. 部署辅助能力
包括但不限于：
- 证书申请
- DNS 验证
- 端口复用
- 端口跳跃
- 网络优化
- 配置导入导出

## 风险提示

这是一个功能密集型一键脚本，通常意味着：
- 逻辑路径多
- 发行版兼容分支多
- 外部依赖多
- 网络与服务写入动作较重

在生产环境使用前，建议逐项核查：
- systemd/service 写入位置
- 防火墙改动
- iptables/nftables 改动
- acme 证书逻辑
- 配置持久化路径
