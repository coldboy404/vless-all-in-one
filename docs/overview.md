# 项目概览

`vless-all-in-one` 是一个面向 Linux 服务器的多协议一键部署脚本，核心入口为：

- `vless-server.sh`

## 当前观察到的定位

从脚本头部与上游说明看，这个项目主要用于：
- 在单机上部署多种代理协议
- 统一管理安装、升级、卸载、订阅与用户
- 同时覆盖 TCP/TLS 与 UDP/QUIC 两类协议栈

## 核心组件分工

脚本当前采用多核心组合方式：
- **Xray**：负责 TCP/TLS 类协议
- **Sing-box**：负责 UDP/QUIC 类协议
- 其他独立组件：用于 Snell、NaïveProxy 等特定协议场景

## 上游文档中提到的主要能力

- 多协议安装与共存
- 用户管理（配额、到期、启停）
- 分流与链式代理
- 订阅生成与聚合
- 端口复用
- 端口跳跃
- CF Tunnel / DNS 验证 / 网络优化

## 维护建议

这个仓库当前更适合作为：
- 代码审查入口
- 二次开发基础
- 功能拆分与模块化重构起点

如果后续要继续整理，建议优先做：
1. 把主脚本按模块拆分
2. 为菜单能力建立函数索引
3. 把高风险操作单独标注
4. 为配置文件写入路径补文档
